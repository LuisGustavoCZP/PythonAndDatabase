# PythonAndDatabase
 Teste e aprendizado de Python com Database
 
## Instalação / Install
### Windows
<ul>
<li><code>python -m venv venv</code></li>
<li><code>.\venv\Scripts\activate</code></li>
<li><code>pip install -r requirements.txt</code></li>
</ul>

### Linux
<ul>
<li><code>python -m venv venv</code></li>
<li><code>source venv/bin/activate</code></li>
<li><code>pip install -r requirements.txt</code></li>
</ul>

## Iniciar / Run
<code>python .</code>
